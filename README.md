<h2 align="center"><strong>Instalação 💻</strong></h2>

```shell script

git clone https://github.com/eutheuzin7/gerador-de-senha-em-php
```

<h2 align="center"><strong>Execução 📂</strong></h2>

```shell script
cd gerador-de-senha-em-php

```
